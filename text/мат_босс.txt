2.1 Стартовые понятия

Определение: Числовая последовательность a(n) при n ->oo сходится к пределу a,
             если по любому e > 0 можно указать такое N, что
             |a(n) - a| < e для всех n > N

Нельзя допустить выпрыгиванию а(n) из e коридора (a-e, a+e).

Определение: Числовая последовательность, имеющая нулевой предел, a(n) -> 0,
             называется бесконечно малой величиной.

Определение: Числовая последовательность a(n) при n ->oo расходится, если по
             любому M > 0 можно указать такое N, что |a(n)| > M для всех n > N.
