IEEE 802.1Q tag Format
    0               1               2               3
    0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7
   +-------------------------------+-------+-+---------------------+
   |             TPID              |       | |       VID (12b)     |
   +-------------------------------+-------+-+---------------------+

TPID = 0x8100 -  IEEE 802.1Q


IEEE 802.1ad tag Format - double tagging

    0               1               2               3
    0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7
   +-------------------------------+-------+-+---------------------+
   |     TPID (0x88A8)             |       | |   outer S-TAG       |
   +-------------------------------+-------+-+---------------------+
   |     TPID (0x8100)             |       | |   inner C-TAG       | Only 0x8100
   +-------------------------------+-------+-+---------------------+


Outer: 0x88A8 (standard), 0x9100, 0x9200
        - 0x9200 Cisco tunnel Q-in-Q

Push Tag --> in case of 0x8100 add OuterTag.
Pop  Tag --> in case QinQ remove OuterTag
